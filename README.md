# Live Webcam Streaming 📹

A real-time video streaming application using Python, Django, and OpenCV.
This project captures video from a local webcam and streams it to a web interface with low latency via an RTSP server.

## 🚀 Features
- **Real-time Streaming:** Uses Motion JPEG (mJPEG) for efficient video delivery.
- **Computer Vision:** Integration with OpenCV for frame processing.
- **Responsive UI:** Clean web interface to view the stream.

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Video Processing:** OpenCV (cv2), FFmpeg
- **Frontend:** HTML5, CSS3, JavaScript
